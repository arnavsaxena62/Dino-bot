# Dino Bot 🦖

A simple C++ bot for the Chrome Dino game using a basic neural network using the GAlib library.

## Files

- `main.cpp` – entry point  
- `dino.cpp / dino.h` – game/bot logic  
- `neuralnet.cpp / neuralnet.h` – neural network code  
- `makefile` – build instructions  

## Build

```bash
make
````

## Run

```bash
./main
```

## About

This was a small project to experiment with:

* C++
* basic neural networks
* game AI / automation
